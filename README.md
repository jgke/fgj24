# Feed the Cats

Cat feeding game, hosted live at https://jgke.fi/games/fgj24/dist

## Running

```
$ npm install # only once
$ npm run dev
```

## Installing on a server

```
$ BASE_URL=/some/www/path npm run build
$ rsync -av dist/ user@some.server:/var/www/html/some/www/path
```
